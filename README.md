# NoticeFix
这是一个用于原生/类原生的**Xposed**模块  
主要目的是修复一些不适配安卓灰度图标规范的通知图标在原生安卓上显示为白块/ 灰块 / 圆块的问题  
同时增加了**自定义图标的功能**   
以及对 **fankes** 的 [Android 通知图标规范适配计划](https://github.com/fankes/AndroidNotifyIconAdapt) 图标的**离线支持**  
理论支持**味精味不重**的原生/类原生**安卓10及以上**
